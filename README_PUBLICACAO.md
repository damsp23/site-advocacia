## Publicação no GitHub Pages
1. Extraia o ZIP.
2. No repositório `site-advocacia`, clique em Add file > Upload files.
3. Envie todos os arquivos e pastas extraídos.
4. Faça o commit.
5. Aguarde o GitHub Pages atualizar e pressione Ctrl + F5 no site.
